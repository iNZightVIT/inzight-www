11 July, 2016

The latest version of iNZight is now available as a beta release, with a new Windows installer, Mapping module, and a whole lot of improvements.

iNZight v3.0 Beta Available

We've been doing a lot of work over the last 6 months, and we're happy to announce that our latest updates to iNZight are ready for you to . While the iNZight you know hasn't gone anywhere, we've made several fundamental changes that we hope will make it easier for you to use.

Many of you have resources that reference certain parts of iNZight, and we have done our best to preserve things as much as possible. In fact, in many places we've made iNZight more consistent, so exploring new parts of iNZight will be just as easy as those you know already.

So, what's new?

  • An improved plot toolbar

    We've reordered some of the buttons, and changed the "Add to Plot", "Remove Additions" and "Inference Information" buttons into icons themselves. It's also now responsive, so when you open modules (see later) the toolbar changes to suit the module.

  • Example datasets

    If you just want to try iNZight, you no longer have to go browsing for the data folder. From the File menu, you'll notice a new "Example Datasets" option, which lets you quickly load data sets to play with.

  • Add to Plot

    We've completely redesigned the add to plot windows, so now you can customise your plot faster without switching between different windows. We've also added more funcionality to it, so you can do more with your plots!

    • Resize points: this now resizes point proportional to the variable, with the option of switching to the old method of scaling all points between a minimum and a maximum size, which is suitable for exaggerating trends.
    • More choice of colours: Finally, you can choose your own colour palette! We've made a few basic ones available, and have also linked the the GUI provided by the colorspace package, which will give you complete control over colouring your plots.
    • Plot symbols: now you can change the plot symbol, and even use it to code variables!
  • Add-on Modules
    • Maps: One of the major new features available in iNZight 3.0 is the Maps module. However, due to the large size of the packages involved, its not included with the download. Instead, it will automatically download and install when you run the Maps module for the first time. This methodology will be used for all future modules added to iNZight.

Windows Installer

Previously, iNZightVIT was supplied as a ZIP archive that needed extracting before it would run. We've moved away from this in preference of a more formal installation which will set things up and place shortcuts on the desktop and in the start menu. We've also included an uninstaller so, if you no longer need iNZight, you can remove it easily.

iNZight VIT breakup

Development of iNZight is moving much faster than VIT, so we've decided to break them up. When you download iNZight, you'll still get VIT, but you'll have two separate Icons for each of them. The main advantage of this is that you can start iNZight directly without first going to the iNZightVIT start menu.

Speaking of VIT, we have a clever student currently working on an online implementation, so anyone with access to a web browser will be able to use it. We'll let you know more when it's ready for use.
