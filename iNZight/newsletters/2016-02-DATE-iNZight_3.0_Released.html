!!!!DATE February, 2016

The latest version of iNZight is now available to download, with a new Windows installer, Mapping module, and a whole lot of improvements.

iNZight v3.0 Released

We've been doing a lot of work over the (Kiwi) summer, and we're happy to announce that our latest updates to iNZight are ready for action. While the iNZight you know hasn't changed (much), we've made several fundamental changes that we hope will make it easier for you to use.

Many of you have resources that reference certain parts of iNZight, and we have done our best to preserve things as much as possible. In fact, in many places we've made iNZight more consistent, so exploring new parts of iNZight will be just as easy as those you know already.

So, what's new?

  • An improved plot toolbar

    We've reordered some of the buttons, and changed the "Add to Plot", "Remove Additions" and "Inference Information" buttons into icons themselves. It's also now responsive, so when you open modules (see later) the toolbar changes to suit the module.

  • Example datasets

    If you just want to try iNZight, you no longer have to go browsing for the data folder. From the File menu, you'll notice a new "Example Datasets" option, which lets you quickly load data sets to play with.

  • Plot features
    • Resize points: this now resizes point proportional to the variable, with the option of switching to the old method of scaling all points between a minimum and a maximum size, which is suitable for exaggerating trends
  • Add-on Modules

    One of the major new features available in iNZight 3.0 is the Maps module. However, due to the large size of the packages involved, its not included with the download. Instead, it will automatically download and install when you run the Maps module for the first time. This methodology will be used for all future modules added to iNZight.

Windows Installer

Previously, we provided a ZIP file of all the necessary components to get iNZight up and running. However, we have discovered that this method can cause some confusing, mostly because Windows can now read and open ZIP files without extracting them, but iNZight has to be extracted for it to run. This meant users had to download and install Unzipping software to get at iNZight.

To counter this, we've created a Windows Installer, an executable (.exe) file. This will install iNZight to a location of your choosing, which is by default the user's Documents folder. Shortcuts are automatically placed on the desktop, and in the Start Menu, so running iNZight is quick and easy.

iNZight VIT breakup

Development of iNZight is moving much faster than VIT, so we've decided to break them up. When you download iNZight, you'll still get VIT, but you'll have two separate Icons for each of them. The main advantage of this is that you can start iNZight directly without first going to the iNZightVIT start menu.

Speaking of VIT, we have a clever student currently working on an online implementation, so anyone with access to a web browser will be able to use it. We'll let you know more when it's ready for use.

iNZight Maps Module

If you've ever wanted to visualise geographical data, but haven't been able to find software that can do it easily, now you can! The Maps module has been designed to make simple geographical data exploration available to all.

There are two parts: plotting points on a map, and shading regions (such as countries) by values of a (numeric, for now) variable (such as population). If you're interesed, read more about it here.

Multiple Response Module

This might not be as interesting as Maps at first, but for a lot of questionaire or survey data, there's no easy way to visually explore the answers to questions where responders can give multiple answers. For example, "Which animals do you own?" will give users several answers to choose from, of which they can choose more than just one answer.

This Multiple response type data requires observing the proportion of "yes"/"no" answers to more than one possible answer at a time. The Multiple Response module makes this easy. You can read more about it here.

And yes, you may have already seen the Multiple Response module in iNZight, but it was not intuitive at all. The new design has been completely reimagined, and follows the iNZight theme to make it easy to use.
